# CLAUDE.md

このファイルは Claude Code がこのリポジトリで作業する際の方針メモです。

## 言語・コミュニケーション

- **ユーザーへの回答は常に日本語で行う。** チャットの説明・報告・確認はすべて日本語。
- コミットメッセージ・PR本文も日本語を基本とする。
- コード内のコメントや識別子は、既存ファイルのスタイルに合わせる（本リポジトリは日本語コメントが中心）。

## プロジェクト概要

- 千葉県松戸市拠点の設備工事会社「合同会社ライフチェンジ」のコーポレートサイト。
- Next.js 14（App Router）/ React 18 / TypeScript / Tailwind CSS。
- 対応エリア: 松戸市・柏市・市川市・流山市ほか近隣。
- 主なサービス: エアコン・給湯器・エコキュート・ガスコンロ・IH・レンジフード・トイレ・水栓・電気工事の交換工事。

## コンテンツ構成

- `content/cases/*.md` … 施工事例（frontmatter付きMarkdown、`data/cases.ts` が読み込み）。
- `content/blog/*.md` … お役立ちブログ記事（frontmatter付きMarkdown、`data/blog.ts` が読み込み、`app/blog/` が表示）。
  - カテゴリは「故障・トラブル」「交換・買い替え」「メンテナンス・季節対策」。
  - 本文のMarkdownは `lib/markdown.ts` の軽量レンダラーで描画（見出し`##`/`###`・箇条書き・番号リスト・太字・リンク・表のみ対応。H1や画像・HTMLタグは使わない）。
- データ定義は `data/`、共通処理は `lib/`、UIは `components/` と `app/`。

## 記事・文章の方針

- 代表「徳永弘之」、現場経験10年以上・年間4,000台以上の施工という強みを誇張せず自然に活かす。
- 料金は必ず「目安」「設置状況により変動」「写真確認後に正式見積り」と添える。
- ガス・電気・高所・水まわりなど危険を伴う内容は安全上の注意を明記し、無理せず専門業者・メーカーへ相談を促す。

## 開発コマンド

- `npm run dev` … 開発サーバー
- `npm run build` … 本番ビルド（記事追加後は必ず通すこと）
- `npm run typecheck` … 型チェック
- `npm run lint` … Lint
